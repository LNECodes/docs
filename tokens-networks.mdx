# **Tokens**

Loop works with any ERC-20 token and SPL token. Please contact us if you'd like to have another token added to your account.

It's important to note that native chain tokens like ETH and MATIC are not ERC-20. However, these tokens have wrapped versions and Loop provides an easy way for merchants to accept them by including a wrapping feature natively in the checkout process. When a customer selects WETH or WMATIC on the native chain, this functionality is displayed.
